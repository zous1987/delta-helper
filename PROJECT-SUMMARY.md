## 📋 项目概述

**目标**: 帮助客服快速判断客户电脑配置是否可接单《三角洲行动》调试服务

**用户**: 5 名客服

**核心价值**:
- ⚡ 10 秒内识别客户配置（原来是手动分析，耗时 2-5 分钟）
- 🎯 准确预测帧率和画质效果
- 📊 自动积累历史案例数据
- 💰 提升接单效率，减少咨询调试师的时间

---

## ✅ 已完成功能

### 1. 🔍 硬件配置识别

**功能**: 粘贴客户配置文本→自动识别硬件型号

**支持识别**:
- CPU（如 i5-12400F、R7 7800X3D）
- GPU（如 RTX 3060、RX 6700 XT）
- 内存（如 16GB、32GB）
- 硬盘（如 512GB NVMe SSD）
- 显示器（如 2K 144Hz）

**测试用例**:
```
输入："你好，我电脑是 i5-12400F，显卡 3060，内存 16G，
      硬盘 512G 固态，显示器 2K 144Hz"

输出：
{
  "cpu": "i5-12400F",
  "gpu": "RTX3060",
  "ram": "16GB",
  "storage": "512GB NVMe SSD",
  "monitor": "2K 144Hz"
}
```

**状态**: ✅ 模拟模式工作正常

---

### 2. 📊 性能预测

**功能**: 根据硬件配置预测游戏性能

**输出内容**:
- 预期帧率范围（如 90-120 FPS）
- 推荐画质等级（低/中/高/极致）
- 是否可接单（✅/⚠️）
- 是否需要调试优化
- 简要分析理由

**测试用例**:
```
输入：{
  "cpu": "i5-12400F",
  "gpu": "RTX 3060",
  "ram": "16GB",
  "storage": "512GB NVMe SSD"
}

输出：
{
  "predicted_fps": "90-120 FPS",
  "recommended_quality": "高",
  "can_run": true,
  "needs_optimization": false,
  "reason": "配置达到游戏推荐标准，CPU 和 GPU 性能均衡，16GB 内存充足"
}
```

**状态**: ✅ 模拟模式工作正常

## 🛠️ 技术架构

### 前端
- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **部署**: Vercel（免费）

### 后端
- **API**: Next.js API Routes
- **AI**: 千问 API（qwen-plus）
- **数据库**: Supabase（PostgreSQL，免费 500MB）

