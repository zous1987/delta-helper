## ⚠️ 常见问题排查

### 问题 1：连接超时（ERR_CONNECTION_TIMED_OUT）

**可能原因：**
- 服务器防火墙未开放
- 阿里云安全组未配置
- 网络不通

**解决方法：**

1. **检查服务器防火墙**（已在服务器上执行）：
```bash
sudo ufw allow 3000/tcp
```

2. **阿里云安全组配置**（重要！）：
   - 登录阿里云控制台
   - 进入 ECS → 安全组
   - 添加入站规则：
     - 端口：3000
     - 协议：TCP
     - 授权对象：0.0.0.0/0（或指定 IP）
     - 优先级：1

3. **测试网络连通性**：
```cmd
ping 10.0.120.14
```

---

### 问题 2：连接被拒绝（ERR_CONNECTION_REFUSED）

**可能原因：**
- 服务器未运行
- 端口绑定错误

**解决方法：**
```bash
# 在服务器上执行
ps aux | grep next-server
# 如果没有进程，重启服务器
cd /home/admin/openclaw/workspace/delta-helper
npm run dev
```

---

### 问题 3：页面空白或加载失败

**解决方法：**
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 使用无痕模式（Ctrl+Shift+N）
3. 换一个浏览器试试

## 🔒 安全建议

### 1. 访问控制（可选）

如果只允许特定 IP 访问，可以在服务器上配置：

```bash
# 只允许公司 IP 访问
sudo ufw allow from 公司 IP to any port 3000
sudo ufw deny 3000/tcp
```
