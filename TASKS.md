**优先级**: 🔴 P0  
**工作量**: 30 分钟  
**价值**: ⭐⭐⭐  
**状态**: ✅ **已完成**（2026-03-19 09:15）

**需求**:
- 预测时自动查询相似历史案例
- 显示 3-5 个最相似案例
- 显示匹配度

**技术方案**:
```typescript
// 在 handlePredict 函数中添加
const fetchSimilarCases = async () => {
  const res = await fetch(
    `/api/cases?similar=true&targetCpu=${config.cpu}&targetGpu=${config.gpu}`
  )
  const result = await res.json()
  setSimilarCases(result.data || [])
}
```
