========================================
  三角洲调试助手 v2.0 - 使用说明
========================================

【快速开始】

1. 首次运行前配置（可选）
   - 如需使用 AI 配置识别功能，需要配置千问 API Key
   - 复制 .env.local.example 为 .env.local
   - 编辑 .env.local，填入你的 QWEN_API_KEY
   - 不配置也可使用，系统会自动使用模拟数据

2. 启动程序
   双击 "start-windows.bat"
   
3. 等待启动
   - 首次运行会自动安装依赖（3-5 分钟）
   - 之后启动只需几秒钟
   
4. 打开浏览器
   访问 http://localhost:3001

【功能说明】
- 客户信息管理
- 配置识别（AI 自动识别）
- 性能预测（调用千问大模型）
- 案例保存
- 补充实际效果（含调试师字段）
- 统计报表（客服排行 + 调试师售后排行）

【数据同步】
当前版本：SQLite 本地版
- 数据保存在本地数据库
- 单电脑使用

【注意事项】
1. 需要安装 Node.js v20+ 
   下载地址：https://nodejs.org/
2. 保持窗口打开，服务在后台运行
3. 按 Ctrl+C 可停止服务
4. 防火墙可能会提示允许 Node.js 访问，点"允许"即可

【获取 API Key】
1. 访问：https://dashscope.console.aliyun.com/
2. 注册/登录阿里云账号
3. 创建 API Key
4. 复制到 .env.local 文件中
