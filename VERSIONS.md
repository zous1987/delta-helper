## 📦 版本规范

**版本号格式**: `v{主版本}.{次版本}.{修订号}`

- **主版本**: 重大功能更新/架构变更
- **次版本**: 新功能添加
- **修订号**: Bug 修复/小优化

**示例**:
- `v1.0.0` - MVP 版本
- `v2.0.0` - 数据持久化 + 客户管理
- `v2.1.0` - 协作流程优化
- `v2.1.1` - Bug 修复
