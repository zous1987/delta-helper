## 📋 客服使用说明

### 新流程

**步骤 1：录入客户信息**
1. 打开首页
2. 填写客户昵称（必填）
3. 填写联系方式（选填）
4. 选择客户来源（选填）
5. 点击"下一步"

**步骤 2：填写配置**
1. 复制客户配置到输入框
2. 点击"🚀 开始识别"
3. 确认识别结果
4. 点击"📊 预测性能"

**步骤 3：查看预测**
1. 查看预测帧率和画质
2. 查看相似历史案例（如有）
3. 点击"💾 保存案例"

## 🐛 常见问题

### 问题 1：页面打不开

**解决：**
```powershell
# 检查服务器是否运行
netstat -ano | findstr :3000

# 如果没有输出，重启服务器
cd C:\delta-helper
npm run dev
```

### 问题 2：案例保存失败

**解决：**
- 检查是否填写客户昵称（必填）
- 检查服务器日志
- 重启服务器

### 问题 3：相似案例不显示

**解决：**
- 首次使用没有历史数据，正常现象
- 保存 5-10 个案例后会自动显示

## 📊 数据说明

### 数据存储

**当前版本：** 内存存储（重启后清空）

**下一版本：** Supabase 数据库（持久化）

### 售后率计算

```
售后率 = 售后单数 / 总单数 × 100%

优秀：< 5%（绿色）
良好：5-10%（黄色）
需改进：> 10%（红色）
```
